# Solution1:
将链表转换为字符串，再转换为数字相加。再用相加的结果构建新的链表。 

PS: 
1. 构建新链表的过程中多次逆序。
2. 注意Corner cases。

# Solution2:
1. 设置进位carry。
2. 遍历两个链表，每次算一位：carry, val = divmod(num1 + num2 + carry, val)。
3. 用val构建新的链表节点：ListNode(val)。

# Solution3:
1. 类似Solution2。每次算一位。
