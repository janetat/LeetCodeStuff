# Solution 1 模拟，暴力法
```
    时间复杂度：O(N ^ 2)
    空间复杂度：O(N)
```
```python
class Solution:
    def lastRemaining(self, n: int, m: int) -> int:
        nums = list(range(n))
        i = 0

        while len(nums) > 1:
            i = (i + m - 1) % len(nums)
            nums.pop(i) # del nums[i]

        return nums[0]
```
