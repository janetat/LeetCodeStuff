# Solution1:
1. 两个隔板的矮的那一个的高度乘以两个隔板的间距就是储水量。
2. 双指针

```python
class Solution:
    def maxArea(self, height):
        left = 0
        right = len(height) - 1
        ans = 0

        while left < right:
            ans = max(ans, min(height[left], height[right]) * (right - left))
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1

        return ans

```
